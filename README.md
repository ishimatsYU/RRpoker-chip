# RRチップ管理システム

## プロジェクト概要

スマートフォン向けに最適化されたポーカールームのチップ管理Webアプリケーション「RRチップ管理」です。顧客は自分のチップ残高を確認し、引き出しは即時反映、入金は管理者承認制で行えます。管理者は全顧客のチップ状況を管理し、チップの増減を自由に操作できます。

## 主な機能

### 顧客向け機能
- **アカウント作成**: 新規ユーザー登録機能
- **ログイン機能**: ユーザー名とパスワードでログイン
- **チップ残高確認**: リアルタイムでの残高表示
- **即時引き出し**: 申請後すぐにチップ残高が減算され、確認画面が表示
- **入金申請**: 管理者の承認が必要な入金申請
- **履歴確認**: 最近の取引履歴とステータス確認
- **RRメンバーチップ表**: 全顧客のチップランキング表示
- **RRトーナメントインマネランキング**: トーナメントでのインマネ合計ランキング
- **トーナメント予定表**: 今後のトーナメント情報表示
- **チップ履歴**: 入出金の履歴確認
- **アカウント削除**: 自己アカウントの削除機能
- **LINE風ナビゲーション**: スマホ最適化の6ボタンナビゲーション

### 管理者向け機能
- **全顧客のチップ管理**: 全顧客のチップ残高一覧
- **入金申請承認**: 入金申請の承認・却下
- **引き出し確認**: 即時引き出しの確認画面（管理者用）
- **チップ自由増減**: 任意の顧客のチップを増減可能
- **ユーザー管理**: 顧客アカウントの削除
- **トーナメント管理**: トーナメントの作成・編集・削除
- **トーナメントプライズ配分**: 入賞者への賞金チップ配分
- **レーキ管理**: レーキ金額の記録と履歴管理

## 使用方法

### 新規アカウント作成
1. トップページで「アカウントを作成」を選択
2. ユーザー名、お名前、パスワードを入力
3. 「作成する」ボタンをクリック
4. 自動的にログイン画面に遷移

### ログイン
1. トップページで「ログイン」を選択
2. ユーザー名とパスワードを入力
3. 「ログイン」ボタンをクリック

### 顧客として使用
1. ダッシュボードで現在のチップ残高を確認
2. 「引き出し」ボタンをクリック → 枚数を入力 → 「引き出す」
3. 確認画面が表示されるので、管理者に見せて確認してもらう
4. 「確認してもらった」ボタンをクリックして完了
5. 「入金」ボタンをクリック → 枚数を入力 → 「申請する」
6. 管理者の承認を待つ
7. 🏆 アイコンをクリックして「RRメンバーチップ表」を確認
8. 自己アカウント削除も可能（確認ダイアログ付き）
9. 下部ナビゲーションから各機能にアクセス

### 管理者として使用
1. ログイン後、承認待ちの入金申請を確認
2. 申請を「承認」または「却下」
3. 「チップ管理」でユーザーを選択し、チップを増減
4. 全顧客のチップ状況を一括確認
5. トーナメントの作成・管理
6. トーナメントプライズの配分
7. レーキの記録と管理

## 特徴

### 🏆 RRメンバーチップ表（ランキング機能）
- 全顧客のチップ残高をランキング表示
- 1位・2位・3位には豪華なバッチとアニメーション
- 1位：金バッチ👑、2位：銀バッチ🥈、3位：銅バッチ🥉
- リアルタイムでランキングが更新される

### 🏆 RRトーナメントインマネランキング
- トーナメントでのインマネ合計をランキング表示
- トーナメントプライズの獲得額を集計
- 上位3位まで特別な装飾とバッチ

### 📅 RRトーナメント予定表
- 今後のトーナメント情報を表示
- 開催日時、受付期間、エントリー料を表示
- 終了したトーナメントは薄暗く表示

### 🚀 即時引き出しシステム
- 画像提出は完全に削除
- 引き出し申請後、すぐにチップ残高が減算
- 確認画面で管理者に見せて即座に対応
- 「引き出す」「キャンセル」の二択に変更

### 👤 新規アカウント作成機能
- アカウント作成かログインかを選択可能
- アカウント作成なしではログイン不可
- 初期チップは0からスタート

### 💰 チップ残高の即時反映
- 引き出し：申請と同時に残高減算
- 入金：管理者承認後に残高増加

### 🔧 管理者機能強化
- チップの自由増減：任意の顧客のチップを増減可能
- ユーザーセレクトボックスから選択
- 増加・減少ボタンで即座に反映

### 🎯 トーナメントプライズ配分
- 開催されたトーナメントの名前を任意入力
- 入賞者（インマネ）をアカウントから選択
- 賞金チップ枚数を入力、該当ユーザーへ即座に追加
- 入賞人数は大会により異なり、1位のみ〜最大5位まで対応
- ユーザーは履歴で、どのトーナメントでいくら賞金を受け取ったかを確認可能

### 💰 レーキ管理機能
- 管理者がレーキ金額を記録
- 総額と履歴数を表示
- レーキ履歴ボタンで過去の記録を閲覧可能

### 🔐 セキュアな管理
- 管理者による入金承認制御
- 引き出しは即時処理だが確認画面で管理
- アカウント削除機能での確認ダイアログ実装

## トラブルシューティング

### ログインできない場合
- ユーザー名とパスワードが正しいか確認
- 大文字・小文字を区別
- ブラウザのキャッシュをクリア

### アカウント作成できない場合
- ユーザー名が既に使用されていないか確認
- すべての必須項目を入力しているか確認
- パスワードが一致しているか確認

### チップ残高が更新されない場合
- ページをリロード
- インターネット接続を確認
- ブラウザのコンソールでエラーを確認

### ランキングが表示されない場合
- 他の顧客アカウントが存在するか確認
- チップデータが正しく作成されているか確認

### トーナメント機能が動作しない場合
- 管理者権限でログインしているか確認
- トーナメントデータが正しく作成されているか確認
- ブラウザのコンソールでエラーを確認
